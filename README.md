# --my-diplom\

# Дипломная работа: Разностные признаки изображений в задачах компьютерного зрения

## Описание проекта
Разработка сверточной нейронной сети для определения секции велосипеда на изображении.

## Результаты
- Лучшая точность на тестовой выборке: **90.94%**
- Количество изображений: **5185**
- Количество классов: **9 секций**

## Структура репозитория
- `ДИПЛОМНАЯ РАБОТА ИЗЕРГИНА.ipynb` - создание датасета и детекция
- `ДИПЛОМ СОЗДАНИЕ НЕЙРОННОЙ СЕТИ.ipynb` - обучение CNN
- `train_results.csv` - результаты на обучающей выборке
- `test_results.csv` - результаты на тестовой выборке

## Как запустить
1. Открыть Google Colab
2. Подключить Google Drive
3. Запустить код последовательно
